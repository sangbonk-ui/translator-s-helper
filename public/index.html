식약처 가축 검역 설문서 — 번역 준수 검수

원문·번역문(.docx)을 좌우 슬롯에 올리고 검수하면, 용어집(verification 폴더) 기준으로 미준수 표현을 좌우 대응으로 표시합니다. 영한병기 파일은 같은 파일을 원문·번역문 양쪽에 각각 올리세요 (원문=영문 열, 번역문=한글 열로 자동 추출).

원문·번역문(.docx)을 올리고 검수 시작을 누르면, 용어집 기준 미준수 문장이 원문(좌)↔번역문(우)으로 매칭되어 표시됩니다. 영한병기 파일은 같은 파일을 양쪽에 올리세요.

verification 폴더의 .docx 또는 .xlsx 용어집을 자동으로 불러옵니다. 아래 셀에서 직접 수정 및 행 추가/삭제가 가능합니다. (term = 자동 검사, note = 수동 검토 기준)
